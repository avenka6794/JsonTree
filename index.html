JSON Tree

A simple library to generate exandable lists from a json objects

Below are some examples of the library

1. Basic

JSON

[
  {
    type: "file",
    name: "document"
  },
  {
    type: "folder",
    name: "Desktop",
    children: [
      {
        type: "file",
        name: "subFile.txt"
      },
      {
        type: "file",
        name: "subFile2.txt"
      },
      {
        type: "folder",
        name: "subFolder2",
        children: [
          {
            type: "file",
            name: "subFolder2File"
          }
        ]
      }
    ]
  }
]
Animation

None ({} - empty object)
